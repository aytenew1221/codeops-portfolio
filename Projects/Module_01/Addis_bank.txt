Day 04  Module1 Project – Addis Bank Account Management System

This program simulates a simple bank account management system using Python object-oriented programming (OOP). It defines an Account class with public attributes (owner and account_number) and a private __balance attribute to protect the account balance. The program provides deposit() and withdraw() methods that validate transactions by rejecting invalid deposits and overdrafts. A read-only @property allows the balance to be viewed safely, while the statement() method displays the account owner's details and current balance in Ethiopian Birr (ETB). The main program demonstrates creating an account, making deposits and withdrawals, handling invalid transactions, and printing account statements.
