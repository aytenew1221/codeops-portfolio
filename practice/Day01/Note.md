Day 01 Notes
Five Terminal Commands

1. pwd

Shows the current directory you are working in.

2. ls

Lists the files and folders in the current directory.

3. cd

Changes from one directory to another.

4. mkdir

Creates a new folder.

5. clear

Clears the terminal screen.

Three Git Commands

1. git init

Creates a new Git repository in the current folder.

2. git add .

Stages all new and modified files so they are ready to be committed.

3. git commit -m "message"

Creates a commit and saves a snapshot of the staged changes with a descriptive message.
